# Agent Instructions

## Project Language Policy

**The working language of this project is English.**

This applies to:
- All source code comments
- All documentation in `docs/`
- All commit messages
- All code review discussions
- All inline documentation and README files

The application itself supports multiple UI languages (English and Chinese) through the Flutter l10n system (`lib/l10n/*.arb`). The Chinese ARB files and generated localizations are **product features** and are exempt from this policy — they must remain in Chinese to serve Chinese-speaking users.

When adding new code or documentation, always write in English. Do not leave Chinese comments, descriptions, or notes in the codebase.
